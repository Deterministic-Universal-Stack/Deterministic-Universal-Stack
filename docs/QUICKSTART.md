# Quickstart

## Install

```bash
npm install
```

## Verify

```bash
npm run verify
```

## Run the Playground

```bash
npm run demo
```

## Replay Through the CLI

```bash
tsx apps/cli/src/index.ts replay
tsx apps/cli/src/index.ts snapshot
tsx apps/cli/src/index.ts persist
```
