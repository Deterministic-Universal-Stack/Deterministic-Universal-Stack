# Quickstart

## Install

```bash
npm install
```

## Verify

```bash
npm run verify
```

## Run the Playground

```bash
npm run demo
```

## Run the Collaborative HTML App

```bash
npm run collab
```

Open:

```text
http://localhost:4321
```

## Run the Peer Communications App

```bash
npm run comms
```

Open:

```text
http://localhost:4545
```

## Run the Forge Demo

```bash
npm run forge
```

## Run the Navigator Demo

```bash
npm run navigator
```

## Run the Social Demo

```bash
npm run social
```

## Run the Runtime Prototype

```bash
npm run runtime-prototype
```

## Run the Distributed Sync Demo

```bash
npm run sync-demo
```

## Run the Benchmark Suite

```bash
npm run bench
```

## Replay Through the CLI

```bash
tsx apps/cli/src/index.ts replay
tsx apps/cli/src/index.ts snapshot
tsx apps/cli/src/index.ts persist
```
